# 🚀 快速启动指南 - V2.0 前端

## 一分钟快速体验

### 1. 启动后端

```bash
# 确保后端服务正在运行
python sniperv52.py
# 或 Go版本
./benz-sniper
```

### 2. 访问新UI

直接在浏览器打开：
```
http://localhost:8001
```

或你的服务器IP：
```
http://your-server-ip:8001
```

### 3. 验证功能

✅ **实时倒计时** - 查看环形进度条是否正常倒计时  
✅ **策略卡片** - 查看实盘/观望状态是否正确显示  
✅ **数据刷新** - 等待2秒，数据应自动更新  
✅ **历史记录** - 滚动查看历史矩阵表格  
✅ **预测推荐** - 点击"复制推荐"按钮测试  

## 📱 移动端访问

1. 确保手机和服务器在同一网络
2. 在手机浏览器输入：`http://[服务器IP]:8001`
3. 添加到主屏幕，获得类原生App体验

## 🐛 问题排查

### 问题：页面空白
**检查**：
1. 浏览器控制台是否有错误（F12打开）
2. 后端服务是否正常运行
3. 访问 http://localhost:8001/api/status 是否返回数据

### 问题：数据不更新
**解决**：
1. 刷新页面（Ctrl + F5 强制刷新）
2. 查看右上角连接状态指示器
3. 检查浏览器控制台网络请求

### 问题：样式异常
**解决**：
1. 清除浏览器缓存
2. 确认 `assets/css/` 文件夹存在
3. 检查网络是否正常（CSS需要从CDN加载Google Fonts）

### 问题：移动端显示问题
**解决**：
1. 使用Chrome移动调试工具测试
2. 确保viewport设置正确
3. 尝试不同的移动浏览器
